Live Safeguards

Current-page preservation signal
Diagnostic only

Route Authority still records when Google strongly aligns a broad or commercial query to the current page, but this signal no longer forces a gate hold by itself.

trust = 0.55 * position_score + 0.45 * impression_score
The signal still uses `avg_position <= 12`, `impressions >= 100`, `trust >= 0.55`, and current-page token preservation, but it is informational rather than a blocking rule.
Top-10 default hold
Ranking protection

Route Authority does not swap a route just because it found another candidate. If the query is already in the top 10, it stays on hold unless there is a strong mismatch override.

Rule: `avg_position <= 10` and no strong wrong-page-type signal.
Reject thresholds
Safety filter

Route Authority rejects when the query is too noisy or too weak to route confidently, unless a specific-intent override is present.

Reject when `noise_penalty >= 30` or `demand_score < 12` without a brand, model, hard-attribute, collection, or very strong fuzzy-match override.
Pass thresholds and mismatch override
Promotion rule

Route Authority only passes when the upside is strong enough, the intent is clear enough, and the route change still looks safe.

Pass when `opportunity_score >= 68` or a strong wrong-page-type mismatch exists, with `intent_clarity >= 55`, `noise < 28`, and `demand >= 30` unless a specific-intent override is active.
