What is a family?

A family is a grouped set of similar GSC queries that share one routing decision. One family can represent many raw Google queries.

When should I use Clear?

Use Clear when the route looks correct and does not need more review. It records that you checked it and found no issue.

When should I use Review?

Use Review when you do not agree with the result. You do not need to decide whether it is a gate issue or a routing issue. Route Authority sends it to admin support and pauses a live block if needed.

Results workflow

Gate status shows whether a route passed Route Authority's checks. Publish status separately shows whether the source page is currently live. The user workflow is simpler than that: if you agree with the result, Clear it. If you do not agree, Review it. A Hold row already means Route Authority did not approve that family.
