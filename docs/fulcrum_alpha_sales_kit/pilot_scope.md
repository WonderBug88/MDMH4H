# Fulcrum Managed Pilot Scope

## Included

- private app install on one BigCommerce store
- catalog sync and attribute normalization
- unresolved mapping review
- cluster-based candidate generation
- high-confidence product-page internal linking
- publish and rollback support
- weekly reporting

## Alpha Constraints

- product-page publishing is the live default
- category publishing is gated beta only
- no auto-publish for unresolved mapping states
- no internal search page targets
- no theme redesign work included

## Suggested Pilot Rollout

1. sandbox install and validation
2. one cluster pilot such as rollaway, luggage, or towels
3. top 25 product sources in production
4. weekly review for 4 to 8 weeks

## Success Metrics

- links published
- source pages affected
- landing-page alignment improvement
- clicks and impressions on pilot clusters
- operator time saved vs manual link curation

## Out Of Scope

- marketplace packaging
- public billing integration
- broad site merchandising changes
- feed management
- content generation outside link blocks
