# Fulcrum Positioning

## What It Is

Fulcrum is a private BigCommerce internal-link orchestration app built for managed SEO pilots.

It does not blindly add links. It:

1. syncs the store catalog
2. normalizes product and category attributes
3. clusters pages by product family
4. scores candidate internal links
5. publishes only intentional, crawlable HTML links

## What Makes It Different

- GSC-informed query-routing engine, not a generic related-products widget
- attribute-aware matching across size, form, pack size, brand, color, finish, and collection
- reviewable recommendations with rationale
- direct BigCommerce metafield publishing
- rollback path without theme rewrites

## Alpha Offer

- private app install
- managed onboarding
- product-page internal linking
- high-confidence auto-publish after readiness checks
- weekly operator reporting
- category beta available later

## Best-Fit Merchant

- medium to large BigCommerce catalogs
- multiple product families and variant-heavy PDPs
- organic search visibility already exists but landing-page alignment is weak
- merchant wants SEO lift without a theme rebuild

## Current Promise

- improve internal relevance and crawl paths on high-confidence product pages
- reduce noisy internal linking
- make recommendations explainable and reversible

## What We Do Not Promise In Alpha

- public marketplace self-serve install
- instant category-page rollout on every store
- automated content rewriting
- guaranteed ranking outcomes
